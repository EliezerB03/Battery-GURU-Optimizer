    About General Settings
    This page contains basic general settings that provide stable optimization.

    CPU Settings: This category offers CPU-related options, such as deep sleep efficiency or heat management.

    Memory Settings: This category offers options related to SWAP virtual memory optimization (Keep these options enabled to improve overall system performance).

    Storage Settings: This category offers settings for improved UFS power efficiency (this options slightly reduces read and write speeds, disable it for better performance).
    Close
    About Freq Settings
    This page contains frequency-related settings to increase or decrease processing speed.

    CPU Freq Settings: This category allows you to increase the speed for Little and Big Clusters (Efficiency and Performance Cores).

    NOTE: Increasing the speed will increase the heat generated by the device and increase instability with an applied undervoltage.
    Close
    About Voltage Settings
    This page contains voltage-related settings.

    Each option offers voltage values based on a margin percentage for CPU, GPU, RAM, Display, Audio, etc.

    NOTE: These settings may make your device unstable or cause freezes if you're not careful.
    Close
